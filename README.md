# #38 Font Spyer

A clever bookmarklet-style tool that reveals font names on any website with a single tap. 

### Core Philosophy
* **1 Tool / 1 Action:** One tap to copy the code, one tap to reveal the font.
* **Performance:** Zero-latency execution as a browser bookmarklet.
* **Privacy:** No data leaves the browser. It purely analyzes the CSS on your screen.

### Features
* **Magic Code Generator:** Instantly provides the Javascript snippet needed for the bookmarklet.
* **On-Demand Analysis:** No need to stay on a specific site—works globally on any URL.
* **Instant Reveal:** Tap any text element to see its font-family pop up in a minimalist black toast.

### Design System
* **Background:** #fff / **Main:** #333 / **Accent:** #eee
* **UI:** Step-by-step card layout with 15px rounded corners.
* **iOS Optimized:** Specialized text-area styling for easy code copying on Safari.
